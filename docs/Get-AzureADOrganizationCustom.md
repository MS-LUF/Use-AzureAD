# Get-AzureADOrganizationCustom

## SYNOPSIS
get all properties of an Azure AD organization

## SYNTAX

```
Get-AzureADOrganizationCustom [<CommonParameters>]
```

## DESCRIPTION

## EXAMPLES

### EXEMPLE 1
```
get all properties of your current organization
```
